### Requirements

**Clone the project and access the folder**

```bash
$ git clone https://github.com/paulohfreire/NOME_DO_REPO.git && cd NOME_DO_REPO
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn start
```
